▸▸ System Capabilities

Features

🔒
E2E Encryption
Every message encrypted with AES-256-GCM. Ephemeral X25519 key exchange. Forward secrecy. Even we can't read your messages — because there is no "we."
X25519 DH + HKDF + AES-256-GCM
🌐
P2P Mesh Network
No central servers. Your messages travel directly between peers through an encrypted mesh. Circuit relay for NAT traversal. DCUtR hole-punching.
libp2p // KadDHT // Circuit Relay v2
👤
Cryptographic Identity
Your identity is an Ed25519 keypair. No email. No phone number. No password. Back it up with a 12-word mnemonic. Recover anywhere.
Ed25519 // BIP39 Mnemonic // TOFU Pinning
📞
Voice & Video Calls
WebRTC calls with DTLS-SRTP encryption. TURN relay fallback. Voice-over-libp2p when all else fails. Your calls, your way.
WebRTC // STUN/TURN // G.711 Fallback
🏰
Hubs
Create community spaces with text channels, voice channels, roles, and invites. Like servers, but decentralized. No one can shut them down.
Categories // Channels // Role Management
👻
Dead Drops
Anonymous onion-routed posts. No identity attached. Proof-of-work spam prevention. Daily rotating network key. Untraceable.
Onion Routing // PoW // Anonymity
📡
P2P Live Streaming
Go live and stream to your network. Mesh-pull distribution — viewers redistribute chunks to each other like BitTorrent. No central streaming server.
WebRTC DataChannels // Mesh-Pull // Stream Chat
