The Decentralized Nature

OpenDescent is a decentralized network. This means:

  • We do not control, moderate, or monitor the content on the network
  • We cannot delete your messages, posts, or data — only you can
  • We cannot ban users from the network — anyone can run a node
  • The network continues to function independently of any single operator

We operate relay nodes to help peers connect, but these are infrastructure — not content hosts.
